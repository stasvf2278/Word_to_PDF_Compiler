Project Name:

Word to PDF Compiler

Description:

This script converts .docx and .doc files to .pdf and then adds the PDFs consecutively in numerical order starting with a '00_' prefix.
A PDF copy of every .docx and .doc file will also be created.

Usage:

Note: Written in Pyton 2.7 and tested in Pyton 3.7

Note: This script was written for Windows.

Note: Change the target folder directly in Ln 9.

Note: Change the output PDF directly in Ln 51.

Installation:

This script is written in the main() feature.
